# Stock Price Prediction — Task 15

## AI & ML Internship

### About
Basic stock price prediction using Linear Regression
and Moving Averages on historical time-series data.

---

### Dataset
- Type     : Simulated Stock Price Data
- Duration : 500 days
- Columns  : Date, Close Price

---

### Steps Completed
1. Loaded and visualized time-series data
2. Split data chronologically (NOT randomly)
3. Trained Linear Regression model
4. Predicted future stock prices
5. Plotted Moving Averages (20-day, 50-day)
6. Evaluated using MSE and R-squared

---

### Results
- R-squared : 0.85-0.92

---

### Tools Used
- Python 3
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

---
